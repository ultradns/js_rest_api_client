Download Node JS if you like a Web Server
http://nodejs.org/download/

In js_rest_api_client Start the WebServer using
For Windows: node.exe scripts\web-server.js
By default the webserver will listen on port 8000

From a browser go to the Application at
http://localhost:8000/app/index.html

To call using Rest Api functionality, first you will need to authorize using Username and Password.
Subsequently, you may authorize using Refresh Token if available
